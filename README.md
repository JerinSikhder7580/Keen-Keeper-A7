#  KeenKeeper — Keep Your Friendships Alive

##  Overview

**KeenKeeper** is a modern friendship management web application that helps users maintain meaningful connections with their friends. It tracks interactions, reminds you when to reconnect, and provides insights into your communication habits.

Built with a clean UI and responsive design, KeenKeeper ensures you never lose touch with the people who matter most.

*  React Router 
*  Tailwind CSS
*  DaisyUI 
*  Recharts 
*  React Toastify (for notifications)

---

##  Key Features

1. **Smart Friendship Tracking**
   Track how long it’s been since you last contacted a friend and stay consistent with your relationship goals.

2. **Interactive Timeline System**
   Automatically logs every interaction (Call, Text, Video) with real-time updates and filtering.

3. **Visual Analytics Dashboard**
   Understand your communication patterns through an interactive Pie Chart.
